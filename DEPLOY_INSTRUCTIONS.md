# Инструкция по деплою на VPS

## Предварительные требования
1. **VPS сервер** с установленным Docker и Docker Compose.
2. **SSH доступ** к серверу (логин/пароль или SSH-ключ).

## Шаг 1: Подготовка сервера
Если Docker еще не установлен на сервере, выполните там эти команды:
```bash
# Обновление системы
apt update && apt upgrade -y

# Установка Docker
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh

# Установка Docker Compose (если нужно, в новых версиях docker он встроен)
apt install docker-compose-plugin
```

## Шаг 2: Настройка деплоя
1. Откройте файл `deploy.sh` в корне проекта.
2. Замените `your_server_ip` на IP-адрес вашего сервера.
3. Если пользователь не `root`, замените `USER="root"` на вашего пользователя.

## Шаг 3: Запуск деплоя
Запустите скрипт из терминала (Git Bash или WSL):
```bash
bash deploy.sh
```

Скрипт автоматически:
1. Скопирует файлы проекта на сервер.
2. Соберет Docker образы.
3. Запустит приложение.

## Шаг 4: Проверка
Откройте в браузере: `http://<ВАШ_IP_АДРЕС>:3000`
